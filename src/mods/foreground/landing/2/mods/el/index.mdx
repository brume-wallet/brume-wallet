## Τι είναι αυτό;

<div className="h-2"/>

Η κορυφαία λειτουργία μας είναι μια ενσωματωμένη υλοποίηση του πρωτοκόλλου δικτύου Tor. Στέλνουμε όλα τα δίκτυα σας μέσω Tor, ώστε οι άνθρωποι στο τέλος του σωλήνα να μην μπορούν να χρησιμοποιήσουν τη διεύθυνση IP σας για να σας παρακολουθήσουν ή να σας λογοκρίνουν.

<div className="h-4"/>

## Πώς λειτουργεί;

<div className="h-2"/>

Όταν το πορτοφόλι σας στέλνει μια δικτυακή αίτηση, επιλέγει ένα τυχαίο κύκλωμα Tor για να στείλει την αίτηση. Αυτό σημαίνει ότι όταν χρησιμοποιείτε πολλές διευθύνσεις πορτοφολιού, κάθε μια θα χρησιμοποιεί διαφορετικό κύκλωμα, ώστε οι άνθρωποι στο τέλος του σωλήνα να μην μπορούν να συνδέσουν τις διαφορετικές διευθύνσεις σας. Η εμπειρία χρήστη είναι ειδικά σχεδιασμένη για να μην ανακτά δεδομένα για πολλές διευθύνσεις ταυτόχρονα.

<div className="h-4"/>

## Κάτι άλλο

<div className="h-2"/>

Επιπλέον, δημιουργήσαμε προσαρμοσμένες υλοποιήσεις του WalletConnect και του Ledger, ώστε να μη διαρρέει καθόλου δεδομένα στο δίκτυο και να μην μπορούν να χρησιμοποιηθούν για να σας παρακολουθήσουν, επιπλέον, επιβεβαιώσαμε ότι διορθώσαμε όλες τις ευπάθειες ασφάλειας και τα προβλήματα απόδοσης που εντοπίσαμε στις αρχικές υλοποιήσεις.
