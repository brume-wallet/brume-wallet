Unser Code ist vollständig unter der MIT-Lizenz lizenziert, es gibt kein Urheberrecht oder ähnliches. ## Reproduzierbar

<div className="h-2"/>

Unsere App verwendet nur öffentliche und weit verbreitete Infrastruktur. Das bedeutet, dass unser Code kein verstecktes Passwort oder Mechanismus zur Nutzung dieser Infrastruktur enthält. Sie können unseren Code frei einsehen und ihn so erstellen, dass er genau den gleichen Bytes entspricht wie die, die wir veröffentlichen.

<div className="h-2"/>

Ein Bot überprüft dies automatisch für jede Veröffentlichung. Das bedeutet, dass selbst wenn unsere Veröffentlichungsmaschine von Malware infiziert ist, die die Bytes, die wir veröffentlichen werden, verändert, der Bot die Manipulation erkennen und die Veröffentlichung verhindern kann.

<div className="h-2"/>

Sehen Sie sich die anderen Abschnitte für weitere Informationen darüber an, wie wir verhindern, dass der Code selbst infiziert wird.
