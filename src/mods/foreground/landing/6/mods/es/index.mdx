## Licencia MIT

<div className="h-2"/>

Nuestro código está completamente licenciado bajo MIT, no hay derechos de autor ni nada similar. Esto significa que en caso de que nuestro equipo sea comprometido, aún puedes usar el código y construirlo tú mismo sin problemas legales. También puedes bifurcar el código y hacer tu propia versión de la aplicación.

<div className="h-4"/>

## Reproducible

<div className="h-2"/>

Nuestra aplicación solo utiliza infraestructura pública y ampliamente disponible. Esto significa que nuestro código no contiene ninguna contraseña oculta o mecanismo para acceder a esta infraestructura. Puedes ver libremente nuestro código y construirlo para que coincida con los mismos bytes exactos que los que publicamos.

<div className="h-2"/>

Un bot hace automáticamente esta comprobación para cada lanzamiento. Esto significa que incluso si nuestra máquina de lanzamiento está infectada por un malware que altera los bytes que estamos a punto de lanzar, el bot aún puede detectar la manipulación y evitar que el lanzamiento se publique.
