## Reproduceerbaar

<div className="h-2"/>

Onze app maakt alleen gebruik van openbare en wijdverspreide infrastructuur. Dit betekent dat onze code geen verborgen wachtwoorden of mechanismen bevat om toegang te krijgen tot deze infrastructuur. U kunt onze code vrij bekijken en bouwen zodat deze exact overeenkomt met dezelfde bytes die we publiceren.

<div className="h-2"/>

Een bot voert automatisch deze controle uit voor elke release. Dit betekent dat zelfs als onze release-machine is geïnfecteerd met malware die bytes wijzigt die we gaan publiceren, de bot nog steeds manipulatie kan detecteren en publicatie kan voorkomen.
