## Reproductible

<div className="h-2"/>

Notre application n'utilise que des infrastructures publiques et largement disponibles. Cela signifie que notre code ne contient aucun mot de passe caché ou mécanisme pour accéder à cette infrastructure. Vous pouvez librement consulter notre code et le construire pour qu'il corresponde exactement aux mêmes octets que ceux que nous publions.

<div className="h-2"/>

Un bot effectue automatiquement cette vérification pour chaque publication. Cela signifie que même si notre machine de publication est infectée par un logiciel malveillant qui altère les octets que nous sommes sur le point de publier, le bot peut toujours détecter la manipulation et empêcher la publication.
