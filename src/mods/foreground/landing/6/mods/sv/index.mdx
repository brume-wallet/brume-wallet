## Reproducerbar

<div className="h-2"/>

Vår app använder endast offentlig och allmänt tillgänglig infrastruktur. Detta innebär att vår kod inte innehåller några dolda lösenord eller mekanismer för att komma åt denna infrastruktur. Du kan fritt granska vår kod och bygga den så att den exakt matchar de bytes som vi publicerar.

<div className="h-2"/>

En bot utför automatiskt denna kontroll för varje utgåva. Detta innebär att även om vår utgivningsmaskin är infekterad med skadlig kod som ändrar bytes som vi ska publicera, kan boten fortfarande upptäcka manipulation och förhindra publicering.
